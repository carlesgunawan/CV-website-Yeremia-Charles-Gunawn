# 💪 Sport Portfolio – Carles Gunawan

Selamat datang di website portfolio saya! 🚀
Website ini dibuat untuk menampilkan perjalanan saya sebagai seorang **sport enthusiast** yang berfokus pada kekuatan, disiplin, dan gaya hidup sehat.

---

## 🧑‍💻 Tentang Website Ini

Website ini merupakan portfolio pribadi berbasis **Laravel + TailwindCSS** yang menampilkan informasi tentang:

* Profil diri (*About Me*)
* Pengalaman dalam dunia olahraga
* Skills (fisik, mental, dan teamwork)
* Projects & aktivitas olahraga
* Achievements (pencapaian)
* Certifications (pembelajaran & pengembangan diri)

Tujuan utama dari website ini adalah untuk mendokumentasikan perkembangan diri sekaligus menjadi inspirasi bagi orang lain untuk menjalani hidup yang lebih aktif dan sehat.
